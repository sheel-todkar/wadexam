PICT B.E. Course Structure

Welcome to PICT B.E. Program

Explore the course structure for all four years:

First Year (FE)

  • Engineering Mathematics I: Covers calculus, linear algebra, and differential equations.
  • Engineering Physics: Basic mechanics, optics, and electromagnetism.
  • Engineering Chemistry: Applied chemistry in engineering.
  • Programming Fundamentals: Introduction to C/C++ programming.
  • Basic Electrical Engineering: Understanding electrical circuits and machines.

Second Year (SE)

  • Data Structures: Covers arrays, linked lists, stacks, and trees.
  • Object-Oriented Programming: Introduction to Java and OOP concepts.
  • Computer Organization: Study of processor architecture and assembly language.
  • Database Management Systems: Fundamentals of relational databases and SQL.
  • Software Engineering: Concepts of software development life cycle.

Third Year (TE)

  • Machine Learning: Basics of AI and ML algorithms.
  • Cloud Computing: Understanding distributed computing and cloud platforms.
  • Operating System: Covers process management, memory allocation, and concurrency.
  • Web Development: Frontend and backend development concepts.
  • Data Science: Analysis, visualization, and predictive modeling.

Fourth Year (BE)

  • Blockchain: Introduction to decentralized applications and smart contracts.
  • Deep Learning: Advanced neural networks and AI models.
  • Mobile Computing: Concepts of mobile application development and cloud integration.
  • Major Project: Industry-based capstone project.
  • DevOps: CI/CD pipelines, automation, and cloud deployment.
